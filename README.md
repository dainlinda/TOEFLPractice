# TOEFL writing practice

## 독립형 유형별 연습
1. [Agree/Disagree]
2. [Two Options]
3. [Multiple Options]
4. [Good idea]

## 연습 방법
1. 질문에 대한 답을 작성한다
  - 30분 이내, 300자 ~ 350자 목표
2. 작성된 답변을 고쳐써본다.
3. 작성한 답변을 검토 받는다.
4. 다음의 내용을 정리한다.
  - 각 표현 혹은 단어에 대한 더 나은 패러프레이징
  - 재사용할만한 고급 표현
  
## 답 작성 순서
1. 각 옵션에 대한 장단점을 3가지 작성한다
2. 더 작성하기 쉬운 쪽으로나의 의견을 정한다
  - 찬성, 반대
  - 선호하는 옵션
3. 정해진 의견에 대한 예시를 생각한다.
  - 경험, 연구 결과, 뉴스 등
4. 서론, 본론(2,3가지), 결론 형식에 맞춰서 러프하게 작성한다
5. 패러프레이징 가능한 문장을 고친다
6. 문법과 오타를 체크한다.
